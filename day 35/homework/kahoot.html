Question 1: What is a function in Python?

Answer: ფუნქცია არის გამოსაყენებელი კოდის ბლოკი, რომელიც კონკრეტულ დავალებას ასრულებს.

def greet(name):
    return f"Hello, {name}!"

Explanation: ფუნქცია greet იღებს პარამეტრს name და აბრუნებს დახელოვნებას.

Question 2: How do you define a function?

Answer: გამოიყენეთ def გასაღები სიტყვა, შემდეგ ფუნქციის სახელით და პარამეტრებით.

def add(a, b):
    return a + b

Explanation: ეს ფუნქცია ემატებს ორ რიცხვს და აბრუნებს ჯამს.

Question 3: What is a return statement?

Answer: ის გამოდის ფუნქციიდან და აგზავნის მნიშვნელობას.

def square(x):
    return x ** 2

Explanation: ეს ფუნქცია აბრუნებს x-ის კვადრატს.

Question 4: How can you pass multiple arguments to a function?

Answer: გამოყავით მათ შორის virgule ფუნქციის განსაზღვრასა და გამოძახებისას.

def multiply(x, y):
    return x * y

Explanation: ეს ფუნქცია იღებს ორ არგუმენტს და აბრუნებს მათ პროდუქტს.

Question 5: What is a default parameter?

Answer: პარამეტრი, რომელიც აქვს დამახასიათებელ მნიშვნელობას, თუ არ არის გაწვდილი ფუნქციის გამოძახების დროს.

def power(base, exp=2):
    return base ** exp

Explanation: ეს ფუნქცია base-ს აყრის exp-ს ძალას, რომელშიც exp-ის მნიშვნელობა 2-ზეა დასანიშნი, თუ ის არ არის გაწვდილი.

Question 6: How do you handle exceptions?

Answer: გამოიყენეთ try და except ბლოკები.

def safe_divide(a, b):
    try:
        return a / b
    except ZeroDivisionError:
        return "Cannot divide by zero."

Explanation: ეს ფუნქცია უსაფრთხოდ იყოფს a-ს b-ზე და აკონტროლებს ნულოვანი დაყოფის შეცდომებს.

Question 7: What is a list comprehension?

Answer: ამრიგად, ხელშესახები სიის შექმნის გამარტივებული გზა.

squares = [x ** 2 for x in range(10)]

Explanation: ეს ქმნის კვადრატების სიის 0-დან 9-მდე.

Question 8: What is the len() function?

Answer: ის აბრუნებს ობიექტში არსებული ელემენტების რაოდენობას.

my_list = [1, 2, 3]
length = len(my_list)

Explanation: len აბრუნებს my_list-ის სიგრძეს, რომელიც არის 3.

Question 9: How can you loop through a list?

Answer: გამოიყენეთ for ციკლი.

for item in my_list:
    print(item)

Explanation: ეს ციკლი ირჩევს თითოეულ ელემენტს my_list-ში და იბეჭდება.

Question 10: What are Python dictionaries?

Answer: ეს არის ცვლადი მონაცემთა ტიპი, რომელიც ინახავს კლავიშ-მნიშვნელობის წყვილებს.

my_dict = {'name': 'Alice', 'age': 25}

Explanation: my_dict ინახავს ორ კლავიშ-მნიშვნელობის წყვილს.
